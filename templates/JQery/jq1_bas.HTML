Selectors

    Syntax: $(selector).action()

    A $ sign to define/access jQuery
    A (selector) to "query (or find)" HTML elements
    A jQuery action() to be performed on the element(s)
    


    JQ selectors test

    <head>
        <script>
            $(document).ready(function(){   // provaid from execute before document is redy
                
                $('p');                     // hide all p element
                $('#test');                 // hide element with id test

                
                
                
                
                

                $('.test');                 // hide element with class test
                $('*');                     // hide all element
                $('[href]');                // hide all href (hiperklink) atribiute

Events

Syntax: $(selector).event(function()action())

Mouse Events
click                               user clicks on the HTML element
dblclick                            user double-clicks on the HTML element
mouseenter                          mouse pointer enters the HTML element
mouseleave                          mouse pointer leaves the HTML element
mousedown                           the left, middle or right mouse button is pressed down the HTML element
mouseup                             the left, middle or right mouse button is released the HTML element
hover                               The first function is executed when the mouse enters the HTML element
                                    and the second function is executed when the mouse leaves the HTML element

on                                  Attach multiple event handlers
Keyboard Events
keypress
keydown
keyup
Form Events
submit
change
focus                               executed when the form field gets focus
blur                                executed when the form field loses focus
Document/Window Events
load
resize
scroll
unload
